# Bengio 2003 : MLP Language Model

Character-level implementation of the architecture from Bengio et al. 2003
on the names dataset. Includes direct connections (eq. 1), mixture model
with trigram, and hyperparameter ablation.

Dataset: names.txt from Karpathy's makemore — https://raw.githubusercontent.com/karpathy/makemore/master/names.txt

Reference: Bengio et al., A Neural Probabilistic Language Model, JMLR 2003
